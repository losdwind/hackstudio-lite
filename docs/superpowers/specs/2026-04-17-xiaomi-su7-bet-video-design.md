## Design System Compliance

Every animation MUST follow `design.md`:
- No solid borders вҖ” use `COLORS.ghostBorder` only when absolutely required
- Glassmorphism on cards: `COLORS.glassSurface` + `backdrop-filter: blur(20-40px)`
- Gradients at 135В°: `GRADIENTS.primaryCTA`
- Ambient glows only: 60-80px blur, 6-10% opacity, tinted via SHADOWS tokens
- Colors from `shared/lib/colors.ts` вҖ” no hardcoded hex/rgba
- Corner radius minimum 12px on containers
- Fonts via `getDisplayFont(lang)` / `getBodyFont(lang)` helpers

Post-implementation audit: scan every new overlay file for hardcoded colors, solid borders, pure-black usage, wrong gradient angles, blur < 20px on glass surfaces.
